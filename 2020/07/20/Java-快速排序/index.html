Java-快速排序（时间复杂度：O(N*logN) ）

简要说明

快速排序由C. A. R. Hoare在1960年提出。

它的基本思想是：通过一趟排序将要排序的数据分割成独立的两部分，其中一部分的所有数据都比另外一部分的所有数据都要小，然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行，以此达到整个数据变成有序。

它采用了一种分治的策略，通常称其为分治法(Divide-and-ConquerMethod)。

最坏运行情况是 O(n²)
