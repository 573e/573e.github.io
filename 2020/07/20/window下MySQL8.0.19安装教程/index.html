window下MySQL8.0.19安装教程

本文最后更新于：2020年7月20日 晚上

window下MySQL8.0.19安装教程

下载MySQL

官网下载地址:

mysql下载

下载完之后解压到本地:

在本目录下新建my.ini文件, 并写入基本配置, 我的是创建过的

设置的安装目录要与解压后的目录保持一致

然后开始安装:

以管理员方式打开命令提示符

进入bin目录

在bin目录下执行:
mysqld –initialize –console

如若出现此提示(没有出现忽略此步骤): 需要安装下面这个

某pan

提取码: gk1a

执行完mysqld –initialize –console是这样:

记号密码, 下面要用

root@localhost: dKc&OEoRq6tR
root: 为账号
dKc&OEoRq6tR: 为初始密码

然后安装MySQL:
mysqld –install

如果出现这种情况:

需要先把mysql的服务删掉
命令:
sc delete mysql

然后再去执行: mysqld –install

安装成功之后, 启动MySQL服务
命令:
net start mysql

然后测试连接MySQL
可以用 Nacicat, 小海豚…等工具进行测试
我这里用的 Navicat

新建MySQL连接
账户密码为开始时的账户和初始密码

连接成功之后
可进行密码修改
密码修改可以用命令修改
alter user ‘root’@’localhost’ identified by ‘newPassword’

我这里用 Navicat 登录之后就提示让修改密码了, 哈哈
安装到此就结束了.
